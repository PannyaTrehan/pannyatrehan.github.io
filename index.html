Home

Welcome to the Sorting Algorithm Visualizer! This website allows you to explore and understand various sorting algorithms through interactive visualizations.

Features

  • Visualize different sorting algorithms including Merge Sort, Quick Sort, Insertion Sort, and Gnome Sort.
  • Learn how each algorithm works with step-by-step explanations.
  • Compare the efficiency and performance of different algorithms.

How to Use

Use the sidebar to navigate to different sorting algorithms. Each page provides a detailed explanation and visualization of the selected sorting algorithm.
